# 🎨UI/UX 리디자인🎨 #

팀 명 : 네잎클로버

팀 원 : 공한나, 송예슬, 안윤지, 하은혜

팀 리더 : 안윤지

목표
  * 예약시스템 관련된 주제 선정 후 앱 버전으로 UI/UX 리디자인
  * 팀원들과의 원활한 소통 및 일관된 디자인

주제 : [아트보다 갤러리](https://www.artbodagallery.com/,"아트보다 갤러리 바로가기")

  * 선정 이유 및 웹 사이트 분석
  * 경쟁사 앱 분석(SWOT)
  * 페르소나
  * 사용자 시나리오
  * 메뉴트리 / 정보구조도
